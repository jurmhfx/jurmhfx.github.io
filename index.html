List of questions for 1.1:

  • What is a markup language?
    • A markup language is a language that describes the content and
    structure of a document by “marking up” or tagging, different document elements.

  • What is XHTML? How does XHTML differ from HTML?
    • Reformulation of HTML 4.01 using XML to provide enforceable standards
    for HTML & allow it to interact with XML languages.

  • What is the W3C? What is the WHATWG?
    • A group of web designers and programmers - World Wide Web Consortium.
    Web Hypertext Application Technology Working Group (WHATWG) was
    formed in 2004 with the mission to develop a rival version to XHTML 2.0, called HTML5.

  • What is a doctype? What is the doctype for an HTML5 document?
    • A processing instruction indicating the markup language used in the document

  • What is incorrect about the following code? Suggest a possible
    revision of the code to correct the error.
    • <p><strong>Curbside Thai now delivers!</p></strong>

    The paragraph tag is before the strong/bold tag. Switching them around
    will fix any errors.
